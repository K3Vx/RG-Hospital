You can change the password at the top of serverHospital.lua. Default is pswd.
You can change the default hospital time and the maximum hospital time at the top of serverHospital.lua.

Commands to type in chat (T):
/hospitalme SECS - Hospitalize yourself, if SECS not given defaults to the default jail time.
/unhospital ems PLAYERID - Unhospitalize the player with PLAYERID (hold up arrow ingame to see) if EMS matches specified hospital password.
/hospital ems PLAYERID SECS - Hospitalizes the player with PLAYERID (hold up arrow ingame to see) if EMS matches specified hospital password. If SECS not given, defaults to the default hospital time.
